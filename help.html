Справочный центр программы

1. Графический интерфейс

  • Добавить/Журнал — открывает окно для ввода новых данных и логов.
  • Выполнить скрипт — загружает файл с командами.
  • Обновить таблицу — синхронизирует данные на экране с диском.

2. Команды скриптов

ADD ресурс; дата; значение

REM поле оператор значение (например: value < 1000)

SAVE путь_к_файлу.txt

3. Архитектура

Программа построена на паттернах Singleton и Publish-Subscribe. Все процессы маршрутизируются в Журнал.
